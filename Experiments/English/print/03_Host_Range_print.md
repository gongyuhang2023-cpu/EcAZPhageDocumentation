# Host Range Determination

**Date:** 2024-11-24 ~ 2024-12-04 | **Operator:** Yuhang Gong | **Status:** Completed

## Objective
Determine host range of 5 phages (R1, R2, R3, W1, W2) against EcAZ-1 and EcAZ-2-OVA through cross-infection testing.

## Materials

**Phages:**
| Name | Original Host | Notes |
|------|---------------|-------|
| R1 | EcAZ-2-OVA | Very small plaques |
| R2 | EcAZ-2-OVA | Turbid plaques |
| R3 | EcAZ-2-OVA | Large, high titer |
| W1 | EcAZ-1 | Small plaques |
| W2 | EcAZ-1 | Medium plaques |

**Test Strains:**
| Name | Code | Notes |
|------|------|-------|
| EcAZ-1 | W | Wild type |
| EcAZ-2-OVA | R | OVA-expressing |

**Reagents:** LB agar plates (1.5%), Soft agar (0.7%), Log-phase culture (OD₆₀₀~0.5)

**Equipment:** Multichannel pipette, 37°C incubator

## Protocol

**Spot Test Method:**
1. Prepare double-layer plates with different host bacteria
2. Serial dilute phages (10⁻¹, 10⁻², 10⁻³)
3. Spot 5uL onto plates
4. Incubate 37°C overnight
5. Observe clear zone formation

**Scoring:** +++ (Clear) | ++ (Semi-transparent) | + (Turbid) | - (No zone)

## Experimental Records
| Time | Operation | Notes |
|------|-----------|-------|
| 11-24~30 | Host range experiment | R/W cross-validation |
| 12-04 | Final confirmation | |

## Results

### Spot Test Results
| Phage | EcAZ-2-OVA (R) | EcAZ-1 (W) |
|-------|----------------|------------|
| R1 | +++ | **−** |
| R2 | +++ | +++ |
| R3 | +++ | +++ |
| W1 | +++ | +++ |
| W2 | +++ | +++ |

### Host Range Classification
| Phage | Host Range | Susceptible Strains |
|-------|------------|---------------------|
| **R1** | **Narrow** | **EcAZ-2-OVA only** |
| R2 | Broad | Both strains |
| R3 | Broad | Both strains |
| W1 | Broad | Both strains |
| W2 | Broad | Both strains |

## Conclusions
1. **4/5 phages have broad host range** (infect both strains)
2. **R1 is the only narrow host range phage** (EcAZ-2-OVA only)
3. R1 specificity may relate to OVA-associated surface receptors

## Notes
- Consider testing more E. coli strains
- R1 narrow host range mechanism warrants investigation
