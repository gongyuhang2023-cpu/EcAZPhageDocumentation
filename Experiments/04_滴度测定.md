## 实验目的

测定5株噬菌体(R1, R2, R3, W1, W2)的滴度(PFU/mL)，为后续实验提供基础数据。

## 实验方案

### 1. 准备工作

**温度控制是关键！**

1. 水浴锅预热至 **50C**
2. 加热模块设定 **45-48C**
3. 微波炉熔化软琼脂，置于50C水浴中冷却至适宜温度
4. 准备好贴标签的底层琼脂平板

### 2. 连续稀释

1. 准备稀释管，每管加入900uL SM缓冲液(或LB)
2. 进行 **1:10 梯度稀释**
3. 建议稀释度：
   - 常规噬菌体：10^-6, 10^-7, 10^-8
   - R3(高滴度)：10^-8, 10^-9, 10^-10

### 3. 双层琼脂法操作

| 步骤 | 操作 | 注意事项 |
|------|------|----------|
| 1 | 向试管加入 **3mL** 熔化软琼脂 | 放回加热模块保温 |
| 2 | 加入 **100uL** 稀释后的噬菌体 | |
| 3 | 加入 **100-200uL** 细菌过夜培养物 | |
| 4 | **涡旋振荡(Vortex)** | **关键步骤！确保充分混合** |
| 5 | 迅速倾注到底层琼脂平板上 | 动作要快，防止凝固 |
| 6 | 轻轻转动平板至完全覆盖 | |
| 7 | 静置凝固 (约20-30分钟) | |
| 8 | 倒置放入37C培养箱过夜 | |

### 4. 计数与记录

**选取噬菌斑数在 30-300 个的平板进行计数**

- 记录噬菌斑数量
- 观察噬菌斑形态(透明/浑浊、有无晕圈、大小)
- 如有重复样，取平均值

### 5. 滴度计算

**公式**:
```
PFU/mL = 噬菌斑平均数量 x 稀释倍数 x 10
```

> 注：x10 是因为只加了100uL(0.1mL)噬菌体，需换算成每mL

**示例**: 10^-6稀释度平均53个噬菌斑
- PFU/mL = 53 x 10^6 x 10 = **5.3 x 10^8 PFU/mL**
