## 实验目的

通过透射电子显微镜(TEM)观察噬菌体形态，确定:
- 头部形态和大小
- 尾部类型和长度
- 噬菌体科/属分类（配合基因组测序结果）

---

## 我需要做的：样品准备

### 第一步：噬菌体扩增

**目标**: 获得足够体积的高滴度噬菌体

| 噬菌体 | 目标滴度 | 目标体积 | 备注 |
|--------|----------|----------|------|
| R1 | >10^9 PFU/mL | ≥50 mL | |
| R2 | >10^9 PFU/mL | ≥50 mL | |
| R3 | >10^9 PFU/mL | ≥50 mL | 本身高滴度 |
| W1 | >10^9 PFU/mL | ≥50 mL | |
| W2 | >10^9 PFU/mL | ≥50 mL | |

**扩增方法**:
1. 准备对数期宿主菌 (100-200 mL)
2. 加入噬菌体 (MOI = 0.01-0.1)
3. 37°C振荡培养至裂解 (3-6h，培养液变清)
4. 离心去除细菌碎片 (8000g, 10min)
5. 0.22μm过滤

---

### 第二步：PEG浓缩

**目的**: 将噬菌体浓缩10-100倍

| 试剂 | 终浓度 | 用量 (每100mL) |
|------|--------|----------------|
| PEG 8000 | 10% (w/v) | 10 g |
| NaCl | 1 M | 5.84 g |

**步骤**:
1. 向过滤后的裂解液加入PEG和NaCl
2. 4°C搅拌溶解
3. 4°C静置过夜（或至少2h）
4. 离心收集沉淀 (10,000g, 20min, 4°C)
5. 用少量SM缓冲液重悬 (1-5 mL)
6. 测定浓缩后滴度

---

### 第三步：氯化铯密度梯度纯化（可选）

> 如果PEG浓缩后纯度不够，需要进一步纯化

**步骤**:
1. 准备CsCl梯度 (1.3-1.7 g/mL)
2. 超速离心 (100,000g, 2-4h)
3. 收集噬菌体条带
4. 透析去除CsCl (SM缓冲液，换液3次)

---

### 送样要求

> ⚠️ 具体要求需与电镜中心确认

| 项目 | 要求 | 备注 |
|------|------|------|
| **浓度** | >10^10 PFU/mL | 越高越好 |
| **体积** | ≥100 μL/样品 | |
| **纯度** | 无明显杂质 | 离心后无沉淀 |
| **缓冲液** | SM或PBS | 无Tris（可能干扰染色） |
| **存储** | 4°C | 不要冷冻 |
