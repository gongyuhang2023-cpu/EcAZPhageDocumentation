## 实验目的

观察并记录5株噬菌体(R1, R2, R3, W1, W2)在各自宿主菌上形成的噬菌斑形态特征。

## 实验方案

### 双层琼脂平板法
1. 准备LB底层琼脂平板
2. 配制软琼脂，冷却至50C左右
3. 加入200uL对数期宿主菌和适量稀释噬菌体
4. 混匀后倒在底层琼脂上
5. 37C培养过夜

### 形态学观察
1. 测量噬菌斑直径 (至少5个噬菌斑)
2. 记录噬菌斑清晰度 (Clear/Turbid)
3. 观察是否有晕环 (Halo)
4. 拍照记录

## 实验结果

### 原始测量数据

| 噬菌体 | 测量值 (mm) | 平均值 |
|--------|-------------|--------|
| R1 | 0.1, 0.1, 0.2, 0.1, 0.2, 0.1 | 0.1-0.2 |
| R2 | 2, 2, 2, 1.5, 1.5, 2 | 1.8+/-0.3 |
| R3 | 4, 3, 2, 3, 4, 3 | 3.2+/-0.8 |
| W1 | 0.5, 0.5, 0.5, 0.5, 0.5, 0.5 | ~0.5 |
| W2 | 2, 3, 2, 2, 2, 2 | 2.2+/-0.4 |

### 形态学总结

| 噬菌体 | 直径 (mm) | 清晰度 | 晕环 (Halo) | 特点 |
|--------|-----------|--------|-------------|------|
| R1 | 0.1-0.2 | Clear | Yes | 极小斑块 |
| R2 | 1.8+/-0.3 | Turbid | No | 浑浊斑块 |
| R3 | 3.2+/-0.8 | Clear | Yes | 最大斑块 |
| W1 | ~0.5 | Clear | Yes | 小斑块 |
| W2 | 2.2+/-0.4 | Clear | Yes | 中等大小 |

### 分类

按斑块大小分类:
- **极小 (<0.5mm)**: R1 (0.1-0.2mm)
- **小 (0.5-1mm)**: W1 (0.5mm)
- **中 (1-2.5mm)**: R2 (1.8mm), W2 (2.2mm)
- **大 (>2.5mm)**: R3 (3.2mm)

### 结论

1. 5株噬菌体的斑块大小存在明显差异
2. **R2是唯一产生浑浊斑块的噬菌体**，可能为温和性噬菌体或溶原性噬菌体
3. R3斑块最大且滴度最高，与分离过程中的观察一致
