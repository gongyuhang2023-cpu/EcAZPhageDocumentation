**🧬 PCR 完整学习（PCR 原理 + PCR SOP）**

**一、PCR 是什么**

**PCR 的本质是：**

**利用温度控制 + 引物特异性，让 DNA 聚合酶只复制你指定的那一段
DNA，并把它指数级放大。**

PCR 并不是"制造 DNA"，而是复制原来就存在的那一段序列。

二、PCR 为什么能只扩增目标片段？（抓住引物这一件事即可）

PCR 的特异性只来自一件事：引物（primer pairing）

引物就像两把定位夹子：

  -----------------------------------------------------------
  引物          位置                           作用
  ------------- ------------------------------ --------------
  Forward       结合到目标序列上游             决定扩增起点
  Primer                                       

  Reverse       结合到目标序列下游（互补链）   决定扩增终点
  Primer                                       
  -----------------------------------------------------------

一对引物把"想要的片段"两端框起来 →
只要温度设置正确，引物就只会贴到它们互补的地方 →
聚合酶只能从引物启动延伸 → 自然只扩那一段

一句话记住：\
PCR 扩什么 ≠ 酶决定，而是 ≡ 引物决定。

**三、PCR 的三步循环**

PCR 每一轮循环只有三步：

  ------------------------------------------------------
  步骤            温度       DNA状态          目的
  --------------- ---------- ---------------- ----------
  Denaturation    94--98°C   双链→单链        拉开DNA
  变性                                        

  Annealing 退火  50--65°C   引物与模板配对   定位片段

  Extension 延伸  68--72°C   聚合酶合成新链   复制片段
  ------------------------------------------------------

这个循环重复 25--35 次 → DNA量呈：

1 → 2 → 4 → 8 → 16 → ... → 2\^30（≈10亿倍）

它不是魔法，是"变性→退火→延伸"三步简单重复造成的指数级增长。

四、PCR 体系怎么配（重点解释 WHY，而不是只给配方）

以 25 μL 体系为例：

  ----------------------------------------------------------------------------
  组分                   体积   用途（重点）
  ---------------------- ------ ----------------------------------------------
  2× Master Mix          12.5   给酶"生命环境"（包含酶、dNTP、Mg²⁺、buffer）
                         μL     

  Forward Primer（10     1 μL   定位起点
  μM）                          

  Reverse Primer（10     1 μL   定位终点
  μM）                          

  模板 DNA               \~1 μL 复制对象

  ddH₂O                  9.5 μL 补足体积
  ----------------------------------------------------------------------------

你需要记住 PCR 配体系的唯一逻辑：

PCR体系决定"能不能扩出来"\
退火温度决定"扩的是不是你想要的"

五、PCR 程序怎么设（理解温度，而不是背温度）

标准程序：

95°C 3 min // 第一次把模板彻底拉开

30 cycles:

95°C 30 sec // 每轮都先拆开双链

55--60°C 30 sec // 引物寻找并结合（关键温度）

72°C 30--60 sec // 聚合酶沿引物复制（1 kb/min）

72°C 5 min // 收尾，补全未延伸完的链

4°C hold

你只要理解一条：

\| 温度越高（接近Tm） \| → 特异性越好（但可能扩不出来） \|\
\| 温度越低 \| → 扩得更容易（但容易出假带） \|

这就是 PCR 优化的全部逻辑。

六、PCR 成败的判断思路（不靠经验也能推理）

PCR 是否成功只取决于 3 个变量：

  -----------------------------
  决定能否扩起来   决定扩不扩
  ---------------- ------------
  模板量           

  -----------------------------

  ----------------------------
  决定扩哪段      决定特异性
  --------------- ------------
  引物 + 退火温度 

  ----------------------------

  ---------------------------------
  决定扩多少       决定最终亮不亮
  ---------------- ----------------
  循环数 +         
  延伸时间         

  ---------------------------------

所以排错永远从这三条检查：

1.  我扩不扩得出来（模板/循环/酶）

2.  我扩的是不是目标（退火温度/引物）

3.  量够不够（循环/时间）

你会发现------这比死记"故障表"容易太多。

七、PCR 的核心心法（这一条你记住就算真的学会）

引物决定方向与范围

温度控制特异性

酶完成逻辑动作

循环制造指数级增长
