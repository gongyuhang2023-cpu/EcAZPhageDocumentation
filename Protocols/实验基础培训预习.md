**实验名称｜目的｜原理｜关键材料｜基础流程**。

拿到菌株/噬菌体 → 分离鉴定 → 扩增纯化 → 滴度测定 → 生长动力学 →
杀菌效果/宿主范围 → 稳定性与PK入门

**噬菌体基础操作**

**1) 斑块实验 / 双层琼脂平板法（Titer by Double-Agar Overlay, DLA）**

**目的**：测定噬菌体滴度（PFU/mL）、观察斑块形态\
**原理**：噬菌体感染宿主产生局部裂解形成透明斑（plaque）；可计数换算滴度\
**材料**：宿主菌对数期培养、噬菌体样品、底层LB-agar、软琼脂（0.5--0.7%）、Ca²⁺/Mg²⁺（常加）\
**流程**：

1.  宿主培养至对数期（OD600≈0.3--0.5）。

2.  噬菌体做10倍系列梯度稀释。

3.  取宿主菌＋噬菌体（各100--200 µL）在37 ℃静置吸附5--10 min。

4.  加入3--4 mL 45--48 ℃软琼脂，混匀倒在预温的底层平板上。

5.  室温凝固，37 ℃倒置培养过夜。

6.  选滴数在30--300之间的平板计数，计算PFU/mL。\
    **视频关键词**："double agar overlay plaque assay phage"

**2) 点滴实验（Spot Test / Host Range Screen）**

**目的**：快速判断样本中是否有裂解活性；筛宿主范围\
**原理**：在已铺好宿主菌的软琼脂层上点加少量噬菌体，观察局部透明区\
**材料**：与DLA类似\
**流程**：

1.  制作"菌草坪"（宿主菌＋软琼脂铺板）。

2.  点加2--10 µL 不同稀释度的噬菌体。

3.  干后37 ℃培养过夜。

4.  观察透明/半透明区；记录相对强度。\
    **视频关键词**："phage spot test host range"

**3) 斑块挑取与纯化（Plaque Picking & Purification）**

**目的**：得到单一克隆的纯噬菌体群\
**原理**：挑取单个斑块至宿主菌中扩增，反复"挑净"除去混合群\
**材料**：无RNAse DNase的移液器头、SM buffer、宿主菌\
**流程**：

1.  在滴度合适、斑块分散的平板，挑取单斑块至100 µL SM buffer中涡旋洗脱。

2.  以该洗脱液进行点滴或DLA确认活性。

3.  重复挑取-扩增-上板 ≥3 轮，直至斑块形态稳定一致。\
    **视频关键词**："plaque picking phage purification"

**\
4) 噬菌体扩增（Liquid Amplification）**

**目的**：制备高滴度裂解液（lysate）\
**原理**：在适MOI下共同培养，让噬菌体在液体中多轮裂解\
**材料**：宿主菌对数期、噬菌体种子、LB或适宜培养基、Ca²⁺/Mg²⁺\
**流程**：

1.  设定MOI（0.01--0.1 常用），混合宿主与噬菌体，37 ℃振荡培养。

2.  观察混浊度下降/澄清（数小时至过夜）。

3.  终止：加氯仿少量（可选）或直接4 ℃离心除菌体细胞残渣。

4.  0.22 µm 过滤收集裂解液，测滴度并保存（4 ℃短期，甘油/SM 低温长期）。\
    **视频关键词**："phage amplification liquid culture"

**5) 滴度测定（Titering, PFU/mL）**

**目的**：标准化样品浓度，后续实验都要用\
**原理/流程**：同#1（DLA），用系列稀释＋平板计数\
**要点**：做技术重复；记录稀释度--平板号--计数原始数据

**\
6) 吸附动力学（Adsorption Assay）**

**目的**：测定噬菌体与宿主结合速率常数；为MOI设计、模型拟合提供参数\
**原理**：在短时间尺度内采样，测自由噬菌体随时间下降的速率\
**材料**：对数期宿主、噬菌体、冰浴/离心\
**流程**：

1.  设定起始浓度与温度，混合后到不同时间点（如0,1,2,5,10
    min）快速离心或过滤分离细胞。

2.  测上清自由噬菌体滴度。

3.  拟合吸附速率常数（k）。\
    **视频关键词**："phage adsorption rate assay"

**7) 单步生长曲线（One-Step Growth Curve：潜伏期/爆发量）**

**目的**：求潜伏期（latent period）、爆发量（burst size）\
**原理**：用低MOI同步感染，短时间间隔取样滴度，得到裂解一周期动力学\
**材料**：对数期宿主、噬菌体、适温摇床\
**流程**：

1.  低MOI感染（0.01左右），短暂吸附后稀释打断再感染。

2.  每隔2--5 min取样测PFU。

3.  曲线中"平台→快速上升→平台"，计算潜伏期和burst size。\
    **视频关键词**："phage one step growth curve"

**\
8) 杀菌曲线 / 时间-杀灭试验（Time-Kill / Killing Curve）**

**目的**：评估不同MOI下对宿主的杀伤效率与复发情况\
**原理**：随时间监测OD600或活菌计数（CFU），并行测PFU变化\
**材料**：摇瓶或深孔板、读数仪\
**流程**：

1.  设定多组MOI（如0、0.01、0.1、1、10）。

2.  0--24 h 定点测OD600与/或CFU、PFU。

3.  比较杀灭速度、复增拐点。\
    **视频关键词**："phage time-kill assay"

**9) 宿主范围与EOP（Efficiency of Plating）**

**目的**：量化同一噬菌体在不同宿主/突变株上的效率差异\
**原理**：在不同菌株上做DLA，计算各菌株上的PFU相对参考宿主的比例\
**材料/流程**：

1.  选定参考宿主（PFU_ref）。

2.  在其他菌株上测PFU_test。

3.  EOP = PFU_test / PFU_ref。\
    **视频关键词**："phage EOP host range"

**\
10) 物理化学稳定性测试（pH/温度/胆盐/血清）**

**目的**：为递送系统与PK解释提供依据\
**原理**：不同条件孵育后比较剩余PFU\
**典型条件**：

- pH 2--3（模拟胃液，短时）；pH 6--8（肠道）

- 胆盐/胆汁模拟液

- 37 ℃/4 ℃/室温；冻融循环

- 血清（热灭活与未灭活）/补体影响\
  **流程**：

1.  设定对照与不同处理（时间×条件）。

2.  处理后立即中和/复温，测PFU。\
    **视频关键词**："phage stability assay pH bile serum complement"

**11) 细胞摄取 / 存活（Gentamicin Protection 或内吞追踪）**

**目的**：初步了解巨噬细胞/上皮细胞对噬菌体的摄取与胞内命运\
**原理**：用抗生素消除胞外噬菌体，残留PFU反映胞内存活；或用标记追踪\
**材料**：巨噬细胞系（RAW264.7/THP-1）或上皮细胞（Caco-2/HeLa等）、温和标记方法（如荧光/噬菌体带报告基因，若有）\
**流程（简版）**：

1.  细胞铺板→加入噬菌体孵育一定时间。

2.  温和洗涤后加gentamicin去除胞外；再裂解细胞测PFU或定量信号。

3.  设定4 ℃（抑制内吞）/37 ℃对照组。\
    **视频关键词**："gentamicin protection assay phage uptake"

**12) 噬菌体DNA提取与基本鉴定（可选）**

**目的**：为后续测序/型别判断做准备\
**原理**：去蛋白/核酸酶处理后提取衣壳内基因组\
**材料**：DNase/RNase 预处理、蛋白酶K、SDS或商用提取试剂盒\
**流程**：标准核酸提取流程＋QC（浓度、纯度、琼脂糖电泳）\
**视频关键词**："phage DNA extraction protocol"

**13) 纯化与内毒素降低（按需求）**

**目的**：细胞实验/动物实验前提高纯度、降内毒素\
**方案**：PEG/NaCl 沉淀→CsCl密度梯度（经典但费时）；或超滤（MWCO 100
kD+）\
**内毒素检测**：LAL试验/替代方法\
**视频关键词**："phage PEG precipitation CsCl gradient phage endotoxin
removal"

**\
极简要点**

- **基本无菌操作**：酒精灯/火焰、移液枪头更换、平板倒置培养

- **软琼脂温度**：45--48 ℃（过热杀噬菌体，过冷易凝固）

- **宿主生长相**：对数期至关重要（OD600 0.3--0.5）

- **记录**：每步记录稀释倍数、体积、温度、时间点、平板编号

- **重复**：关键数据做≥3个技术重复

- **保存**：裂解液短期4 ℃，长期加入甘油或在SM中低温保存

- **生物安全**：按实验室SOP（E. coli菌株分级、噬菌体废弃物灭活方式）

**入门阅读（你明天可扫一眼的"概念地图"）**

- 噬菌体术语：MOI、PFU、burst size、latent period、adsorption constant

- 宿主--噬菌体互作：LPS/受体决定的宿主范围、抗噬菌体机制（CRISPR-Cas、R-M、受体变异）

- 影响活性的环境因素：pH、温度、盐、胆汁、补体

- 体外到体内的桥接：体外滴度≠体内有效浓度；清除（肝脾MPS、肾、抗体）与递送屏障是Project
  2/3的切入点
