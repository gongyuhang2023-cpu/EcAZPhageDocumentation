**🧬 从污水分离噬菌体 SOP（预处理 → 富集 → 验证 → 纯化 → 扩增）**

**0) 准备与总体思路**

- **宿主**：对数期 E.
  coli（OD600=0.3--0.5），至少1株，最好2--3株不同受体背景（提高命中率）。

- **缓冲液**：SM buffer（100 mM NaCl, 50 mM Tris-HCl pH7.5, 10 mM
  MgSO₄，0.01%明胶可选）。

- **过滤耗材**：**0.22 μm PES 或 PVDF**（避免尼龙
  Nylon，易吸附噬菌体）。可先用 0.45 μm 预滤去大颗粒。

- **添加剂**：CaCl₂ / MgSO₄ 各 1--5 mM（促进吸附）。

- **培养基**：LB（液体与平板），Top agar 0.5--0.7%。

- **原则**：**先去掉污水里的细菌（离心/过滤），再用"你养的对数期宿主"去富集污水中的噬菌体**。

------------------------------------------------------------------------

**1) 样本预处理（把"污水"变成"可能含 phage 的澄清液"）**

1.  **收集**：新鲜污水 50--500 mL（装棕色瓶，避光，4℃短期保存）。

2.  **低速离心**：5,000--8,000 g，10--15 min，去大颗粒与细胞沉淀。

3.  **取上清预滤**：可先 0.45 μm 去粗颗粒（可选，但常见），再**0.22 μm
    PES/PVDF 过滤**获得澄清液。

    - 目的：**去除细菌**（避免把污水细菌带进富集体系）。

4.  （可选）**加入SM**：若体积需要，等体积或按比例换成/补至 SM
    buffer，有利于后续稳定。

⚠️
不建议对环境样本一上来加氯仿（CHCl₃）：会杀菌，但也可能影响某些脆弱噬菌体；通常等拿到阳性再考虑用。

输出：**污水滤液（无细菌，可能含 phage）**

------------------------------------------------------------------------

**2) 富集（让 phage 在你的宿主上"被动放大"）**

为每一株宿主分别做一管富集（并做空白对照）。

**配比建议（单管 10--50 mL 规模）**

- 新鲜 LB：9 份

- **污水滤液**：1 份（起步 1:10 体积比；也可 1:1 或 1:5
  做平行，增加命中率）

- Ca²⁺/Mg²⁺：终浓度 1--5 mM

- 接种**对数期宿主**到 **起始 OD600≈0.05--0.1**（保证培养中持续处于
  log）

**培养条件**

- 37℃，200--250 rpm，**6--18 h**（常见 8--12 h 最合适）

- 观察：先浑浊→后可能变清（出现裂解）

**收尾处理**

- 12,000 g，10 min，去除碎屑

- **0.22 μm 过滤** → 得到\*\*"富集裂解液（enriched lysate）"\*\*

输出：**富集后的噬菌体裂解液**

✳️ 逻辑：样本中若有能感染你宿主的
phage，它会在这一步被指数级放大，变成可检测水平。

------------------------------------------------------------------------

**3) 快速验证（Spot test）与确证（Plaque assay）**

**3.1 Spot test（先筛）**

- **制备菌膜**：对数期宿主 + soft agar（或直接涂布成 lawn）。

- 在表面**滴加富集裂解液** 2--10 μL（做 10⁰--10⁻³ 多梯度点几滴）。

- 吸干 5--10 min → 倒置 37℃ 过夜。

- **判读**：透明区/抑菌圈 = **阳性**；+++ / ++ / + / -- 记录强弱。

一旦有阳性 → 立刻做 plaque assay 确认。

**3.2 Plaque assay（确证 + 估滴度）**

- 用"富集裂解液"做 10 倍系列稀释（10⁻¹～10⁻⁸）。

- 与对数期宿主混合（各 100 μL），37℃吸附 5--10 min。

- 加入 3 mL 45--50℃ top agar，倒板，过夜。

- **计数** 30--300 斑块的板 → 计算 PFU/mL。

输出：**确认有噬菌体 & 得到初始滴度**

------------------------------------------------------------------------

**4) 斑块纯化（确保单克隆）**

- 从清晰、孤立的单个斑块（plaque）挑取 → 100 μL SM buffer 震荡洗脱。

- 以此为"phage 源"，重复 **Plaque assay → 再挑斑**，**至少 2--3 轮**。

- 目的：**去掉混合 phage**，得到遗传上尽量一致的**单克隆噬菌体**。

输出：**纯化的 phage 克隆**

------------------------------------------------------------------------

**5) 扩增与保存（得到可用 stock）**

- **液体扩增**（你已学过）：

  - 50 mL LB（250 mL 瓶）、起始宿主 OD600 0.05--0.1，MOI
    0.01--0.1，37℃摇 4--18 h 直至澄清。

  - 12,000 g 离心 → **0.22 μm 过滤** → **得到高滴度 lysate（10⁹--10¹¹
    PFU/mL）**。

- **保存**：

  - 短期 4℃（SM buffer）避光分装；

  - 中期可 +0.1--1% 氯仿（注意个别 phage 例外）；

  - 长期 -80℃（加保护剂）/ 冻干；

  - 记录表：Phage ID / 宿主 / 日期 / 滴度 / 保存条件。

------------------------------------------------------------------------

**关键问答（帮你完全吃透）**

**Q1：能不能不去除污水细菌，直接"整体扩增"？**\
**不建议。污水细菌会与实验室宿主竞争、产生不可控变量、甚至带入病原风险。标准流程是离心+0.22
μm 过滤**后，只保留 phage 去感染你自己培养的宿主。

**Q2：仅用"非尼龙 0.22 μm 滤膜过滤"就能得到 phage 吗？**\
**能得到"可能含 phage
的滤液"**，但**不代表数得出斑**。通常还需要**富集**（跟对数期宿主共培养）把
phage 放大到可检测水平，再做 spot/plaque。

**Q3：滤膜材质为什么要避开尼龙？**\
尼龙容易**吸附**某些 phage，导致"假阴性"。**PES / PVDF**更稳。

**Q4：富集时间多长合适？**\
常规 **6--12 h**。\<3 h 常常太早，\>18 h 要注意二次污染或 phage
稳定性下降。

**Q5：如何提高成功率？**

- 准备 **2--3 株不同宿主**平行富集；

- **样本新鲜**、来源优（WWTP 进水口＞普通湖水）；

- **全程避光、4℃暂存**；

- **log phase 宿主** + **Ca/Mg**；

- 先 **Spot test**，再 **Plaque**。

------------------------------------------------------------------------

**迷你速查版（贴在台面）**

污水 → 低速离心 → 0.22μm(PES/PVDF) → 得滤液

滤液 + log期宿主 + Ca/Mg → 37℃摇 6--12h → 0.22μm

→ Spot test（阳性） → Plaque assay（确证+滴度）

→ 挑单斑纯化2--3轮 → 液体扩增 → 过滤 → 保存（SM, 4℃/−80℃）
