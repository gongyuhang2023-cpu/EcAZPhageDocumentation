**🧪 Spot Test（滴点实验，裂解谱 / 宿主范围检测）**

这一项是噬菌体实验里**最轻便、最快速、最常用的筛查方法**，你未来一定会高频使用它。

------------------------------------------------------------------------

**✅ 一、Spot Test 用来做什么？（一句话理解）**

**Spot Test 用来快速判断噬菌体是否能裂解某个细菌，以及裂解强度的差异。**

也就是说，如果你有一堆 *宿主候选菌 / 突变株 /
临床分离株*，你不用一块块做 Plaque Assay，\
只需要 **在同一块 plate 上滴几个小点** ------ 就能知道：

- 哪些菌对这个 phage **敏感**

- 哪些菌 **不敏感**

- 哪些介于中间（弱敏感）

因此 Spot test 的核心价值是：

  ------------------------------------------------------
  **功能**                              **Spot
                                        test表现**
  ------------------------------------- ----------------
  **宿主范围（Host range）快速筛选**    哪些菌能被裂解

  **突变株 vs 野生株对比**              哪株更抗 phage

  **免疫/耐药/宿主特异性实验初筛**      是否存在差异性

  **是否继续做 plaque                   判断优先顺序
  assay/动力学实验的依据**              
  ------------------------------------------------------

------------------------------------------------------------------------

**✅ 二、与 Plaque Assay 的区别（你脑中要有的清晰定位）**

  ---------------------------------------------------------------
  **项目**   **Spot test**                  **Plaque assay**
  ---------- ------------------------------ ---------------------
  目的       快速判断裂解能力               精准测滴度 & 观察斑形

  工作量     **小**                         大

  灵敏度     中                             高

  结果       \+ / -- / ++ / +++（裂解等级） 清晰斑块（PFU
                                            可计数）

  场景       多株筛选、初步验证、宿主范围   后续实验、正式数据
  ---------------------------------------------------------------

一句话总结：

**Spot test = 快速筛查，Plaque assay = 精准定量**

------------------------------------------------------------------------

**✅ 三、实验原理**

对数期菌先铺成一层均匀菌膜（bacterial lawn）\
→ 再将噬菌体样品**滴加在表面**\
→ 如果 phage 能感染该菌 → 局部出现"抑菌圈 / 透明区 / 斑点"\
→ 强弱不同 → 表现裂解等级

------------------------------------------------------------------------

**✅ 四、实验材料（Checklist）**

  -------------------------------------------------------
  **你需要准备的**                **备注**
  ------------------------------- -----------------------
  LB agar 平板（1.5%）            底板

  对数期宿主菌                    必须 log phase

  soft agar（或直接菌膜法）       两种方案都行

  噬菌体样品（原液或稀释液）      最好多梯度，例如
                                  10⁰--10⁻⁴

  无菌枪头、吸头、酒精灯/超净台   无菌操作

  37°C培养箱                      过夜
  -------------------------------------------------------

✅ 你会发现材料清单和 Plaque assay 非常接近

------------------------------------------------------------------------

**✅ 五、Spot Test SOP（两种方法，一目了然）**

你预习时只需要记住"**菌先铺好 → 酶再滴上去**"。

**方法 A：软琼脂滴点法（Layered method，结果最清楚） ✅推荐**

1）用对数期菌制备 lawn（和 plaque assay 的 top agar 步骤类似）

- 对数期菌 100 μL + soft agar 3 mL → 倒平板 → 凝固

2）在表面**滴加噬菌体溶液**（2--10 μL/点，不要太大）

3）室温吸收 5--10 min（滴点完全干、不要流）

4）倒置 → 37℃ 过夜培养

**第 2 天观察：点状透明区/浑浊抑制区 → 即为裂解阳性**

------------------------------------------------------------------------

**方法 B：菌膜滴点法（更快更省事）**

1）用对数期菌 **Spread plate 或 swab** 在平板上涂一层菌膜\
2）待 5--10 min 风干\
3）滴 phage（2--10 μL/点）\
4）同样过夜培养

结果判断与方法 A 一致\
区别是 **不需要 soft agar**，但斑形会稍模糊

------------------------------------------------------------------------

**✅ 六、结果判读（最好记住这一张表）**

  ------------------------------------------------------
  **裂解情况**   **板面表现**         **代表意义**
  -------------- -------------------- ------------------
  **+++**        大片透明 /           强裂解，宿主敏感
                 完全抑菌区           

  **++**         较清晰透明斑         中等裂解

  **+**          浅色抑制圈 / 雾状    弱裂解

  **--**         无变化               宿主不敏感或抗性
  ------------------------------------------------------

未来你做宿主范围图（Host range heatmap）就是根据这个等级来的

**✅ 七、常见问题（你预习阶段提前知道就能避坑）**

  ------------------------------------------
  **问题**         **常见原因**
  ---------------- -------------------------
  滴点糊成一片     点太大或未风干

  lawn             涂布不均匀
  一块深一块浅     

  弱裂解看不清     phage
                   滴度太低，需做多梯度

  板面很多杂菌     无菌操作不到位
  ------------------------------------------

**✅ 八、一句话总复盘**

对数期菌做菌膜 → phage 滴点 → 过夜 →
有透明区则为裂解，可判强弱，可判宿主范围。
